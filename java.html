Fundamentos de Programación en Java

Introducción a Java

  • Lenguajes de programación: Idiomas artificiales para expresar cálculos y procesos ejecutados por ordenadores.
  • Algoritmos: Conjunto ordenado y finito de operaciones para resolver un problema.
  • Java: Lenguaje orientado a objetos desarrollado por Sun Microsystems (ahora Oracle), destacado por su portabilidad ("Write Once, Run Anywhere").

Plataforma Java

  • Bytecode: Lenguaje intermedio al que se compila el código Java.
  • JRE (Java Runtime Environment): Entorno de ejecución que incluye la JVM (Java Virtual Machine).
  • JDK (Java Development Kit): Conjunto de herramientas para desarrollar aplicaciones Java.

Proceso de desarrollo

  • Edición: Escritura del código fuente en un archivo .java.
  • Compilación: Conversión del código fuente a Bytecode (archivo .class).
  • Ejecución: Interpretación del Bytecode por la JVM.

Estilo de programación

  • Legibilidad: Código claro y fácil de entender.
  • Corrección: El programa debe cumplir con los requisitos especificados.
  • Eficiencia: Optimización de recursos (tiempo y memoria).

Estructura del Programa

Un Programa típico de Java se compone de una o más clases. Cada clase puede contener métodos y variables. Aquí tienes un ejemplo básico:

                
public class HolaMundo {
    public static void main(String[] args) {
        System.out.println("Hola, Mundo!");
    }
}

Estructura de un programa Java

Elementos básicos

  • Comentarios
    • /* ... */ (bloque)
    • /** ... */ (documentación)
    • // (línea)
  • Identificadores
    • Nombres de variables, métodos y clases. Reglas:
    • Comienzan con letra o _.
    • Distingue mayúsculas/minúsculas.
    • No pueden ser palabras reservadas.
  • Variables
    • Declaración: tipo nombre; (ej. int edad;).
    • Tipos primitivos: byte, short, int, long, float, double, char, boolean.
    • Literales: Representación de valores (ej. 10, 3.14, 'A', "Hola").
  • Operadores
    • Aritméticos: +, -, *, /, %.
    • Relacionales: >, <, ==.
    • Lógicos: &&, ||, !.
  • Expresiones
    • Combinaciones de operadores y valores (ej. (a + b) * c).

Ejemplo básico

                
public class HolaMundo {
    public static void main(String[] args) {
        System.out.println("Hola Mundo");
    }
}

Clases y objetos

Definiciones

  • Clase: Plantilla que define atributos y métodos de un objeto.
  • Objeto: Instancia de una clase con estado (valores de atributos) y comportamiento (métodos).

Elementos de una clase

  • Atributos: Variables que almacenan el estado del objeto. 
                            
public class Vehiculo {
    private String matricula;
    private String marca;
}
  • Métodos: Funciones que definen el comportamiento. 
                            
public String getMarca() {
    return this.marca;
}
  • Constructores: Métodos especiales para inicializar objetos. 
                            
public Vehiculo(String matricula, String marca) {
    this.matricula = matricula;
    this.marca = marca;
}

Ejemplo de objeto

                
Vehiculo miCoche = new Vehiculo("1234ABC", "Toyota");
System.out.println(miCoche.getMarca()); // Salida: Toyota

Extensión de clases

Herencia

  • Superclase/subclase: Una clase hereda atributos y métodos de otra. 
                            
public class Turismo extends Vehiculo {
    private int puertas;
}
  • Polimorfismo: Un objeto puede tomar muchas formas (ej. un Turismo es un Vehículo).

Composición

  • Una clase contiene objetos de otras clases. 
                            
public class VehiculoAlquilado {
    private Cliente cliente;
    private Vehiculo vehiculo;
}

Estructuras de control

Selección

  • if-else: Ejecuta bloques según una condición. 
                            
if (edad >= 18) {
    System.out.println("Mayor de edad");
} else {
    System.out.println("Menor de edad");
}
  • switch: Selección múltiple. 
                            
switch (dia) {
    case 1: System.out.println("Lunes"); break;
    default: System.out.println("Otro día");
}

Repetición

  • while: Repite mientras la condición sea verdadera. 
                            
while (i < 10) {
    System.out.println(i);
    i++;
}
  • for: Repite un número conocido de veces. 
                            
for (int i = 0; i < 10; i++) {
    System.out.println(i);
}

Estructuras de almacenamiento

Arrays

  • Declaración: 
                            
int[] numeros = new int[5];
String[] nombres = {"Ana", "Juan"};
  • Acceso: 
                            
System.out.println(numeros[0]); // Primer elemento
  • Arrays multidimensionales: 
                            
int[][] matriz = {{1, 2}, {3, 4}};
System.out.println(matriz[0][1]); // Salida: 2
