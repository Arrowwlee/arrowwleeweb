Conceptos básicos

Python utiliza una sintaxis simple y directa. Aquí tienes un ejemplo básico:

                
print("Hola, Mundo!")

Python para Todos

Introducción a Python

Python es un lenguaje de programación fácil de aprender y potente. Su sintaxis clara lo hace ideal para principiantes.

                
print("Hola, Mundo")  # Esto imprime un mensaje en la pantalla

Tipos de datos básicos

Python maneja diferentes tipos de datos, como números, textos y booleanos.

                
entero = 10          # Número entero
flotante = 3.14      # Número decimal
texto = "Python"     # Cadena de texto
verdadero = True     # Valor booleano

Listas y diccionarios

Las listas almacenan colecciones ordenadas, y los diccionarios guardan datos con claves y valores.

                
frutas = ["manzana", "banana", "naranja"]  # Lista
persona = {"nombre": "Ana", "edad": 25}     # Diccionario

Condicionales y bucles

Permiten controlar el flujo del programa con decisiones y repeticiones.

                
if edad >= 18:
    print("Mayor de edad")
else:
    print("Menor de edad")

for fruta in frutas:
    print(fruta)

Funciones

Bloques de código reutilizables que realizan una tarea específica.

                
def saludar(nombre):
    return f"Hola, {nombre}"

print(saludar("Carlos"))

Programación orientada a objetos (POO)

Python permite crear clases y objetos para organizar el código de manera eficiente.

                
class Persona:
    def __init__(self, nombre):
        self.nombre = nombre

    def presentarse(self):
        print(f"Soy {self.nombre}")

p = Persona("Luisa")
p.presentarse()

Trabajar con archivos

Python puede leer y escribir archivos fácilmente.

                
with open("archivo.txt", "w") as f:
    f.write("Guardando texto")

Expresiones regulares

Útiles para buscar y manipular patrones en textos.

                
import re
texto = "Python es genial"
resultado = re.search("Python", texto)
print(resultado.group())  # Imprime "Python"

Bases de datos con SQLite

Python incluye soporte para bases de datos SQLite sin necesidad de servidor externo.

                
import sqlite3
conexion = sqlite3.connect("ejemplo.db")
cursor = conexion.cursor()
cursor.execute("CREATE TABLE usuarios (nombre TEXT, edad INTEGER)")
conexion.close()

Documentación con docstrings

Los comentarios especiales que explican el código.

                
def suma(a, b):
    """Esta función suma dos números."""
    return a + b

print(suma.__doc__)  # Muestra la documentación

Conclusión

Python es un lenguaje versátil, ideal para principiantes y expertos. Su legibilidad y amplia comunidad lo hacen perfecto para desarrollo web, análisis de datos, automatización y más.

                
print("¡Aprende Python y crea algo increíble!")
